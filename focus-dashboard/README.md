# Focus Dashboard

Odaklanma ve verimlilik panosu - Inside Out çalışma teması ile

## 🚀 Hızlı Deploy

### 1. GitHub'a Push Et
```bash
# Git repository oluştur
git init
git add .
git commit -m "Initial commit - Focus Dashboard"

# GitHub'a bağla
git remote add origin https://github.com/KULLANICI_ADIN/focus-dashboard.git
git branch -M main
git push -u origin main
```

### 2. Vercel Deploy
1. [Vercel.com](https://vercel.com) → New Project
2. GitHub repository seç
3. Deploy butonuna tıkla

**✅ 30 saniyede live!**
